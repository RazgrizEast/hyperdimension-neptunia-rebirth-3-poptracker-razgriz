A PopTracker for the Archipelago implementation for Hyperdimension Neptunia Re;Birth 3

I'm reasonably sure the logic is all correct (barring any errors or misunderstandings I surely made). You can ping me on Discord (I am in the Archipelago server as Razgriz) to let me know of any errors or suggestions.


**TO USE:** Simply download the .zip file and put it in your Poptracker packs folder.

-The tracker has various layers of maps (an Overworld that sees everything, an Overview map for each area, and then individual dungeon maps)

-Locations are split between Gather/Treasure in one box and Monsters in another, per dungeon.


**KNOWN LIMITATIONS:** This tracker will not automatically switch tabs as you move through the game. Doing that would require the mod itself to send location info to the Archipelago server's slot date. Even then, I'd need to figure out how to take that data and build it into the tracker, which I still don't know how to do. So for the most part, use the tracker manually to quickly glance at where you need to go.

I also have not figured out how to keep monsters that are in multiple dungeons from showing as available if you can access another dungeon they are also in. Obviously, the dungeon you have access to is the one you need to go to.
